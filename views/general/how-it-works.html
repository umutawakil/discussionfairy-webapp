How it Works


Create a discussion using the mobile app by recording yourself talking for up to 60 seconds.


When you create your discussion you give it a unique ID and a corresponding link which can be shared online.


People can respond to your comment with their own 60 second recording and anyone on the web can listen to the discussion using the link you created.
